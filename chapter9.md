# 第9章 - 使用pdftk

*Pdftk*是一个基于iText库的多平台命令行工具（在第116页的“iText for Java和C＃”中有描述）。 
它具有合并，拆分和标记文档以及设置和读取元数据的功能。

## Command Line Syntax
*Pdftk*有一个不寻常的命令行界面，其中元素通常必须以特定顺序出现。
我们可以按照指定的顺序将它们分成四组：

* 1.输入文件或文件，以及可能的输入密码。
* 2.操作及其所需的任何参数。
* 3.输出和任何输出密码和权限。
* 4.杂项输出和其他选项。

完整的详细信息可以在*pdftk*的手册中找到 - 在本章中，我们仅提供示例所需的子集。
